# Amusement-Park-Eligibility-Checker-C-
This program asks the user for age and height (in inches), then uses conditionals to determine if the user can ride.
### Rules
- Age must be 12 or older
- Height must be 48 inches or taller

Skills Learned:

• Using cin and cout for user input and output

• Declaring and using int variables

• Writing if / else conditional logic

• Validating age and height using logical operators (&&)
